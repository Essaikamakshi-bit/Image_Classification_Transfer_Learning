Image Classification using Transfer Learning
This project implements an image classification model using transfer learning with TensorFlow/Keras.

Objective
To build a small deep learning image classifier using a pretrained model.

Dataset
CIFAR-10 dataset

Tools & Libraries
Python
TensorFlow 2.19.0
Keras
Google Colab
Features
Transfer learning
Data augmentation
Training & validation accuracy/loss plots
Prediction with confidence score
How to Run
Open the notebook in Google Colab
Run all cells sequentially
